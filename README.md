## Introduction
`MyShell`은 Linux 환경에서 간단한 명령어를 실행하기 위한 사용자 정의 셸입니다.

## Features
- **File Management Commands**
  - `ls`: 디렉토리 내 파일 목록을 출력합니다.
  - `cat`: 파일의 내용을 출력합니다.

- **Navigation Commands**
  - `pwd`: 현재 작업 디렉토리 경로를 출력합니다.
  - `cd <directory>`: 작업 디렉토리를 변경합니다.
